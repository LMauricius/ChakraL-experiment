    • Comparison operators

    Usage

    Comparison operators are operators usually used in math but also repurposed for other use-cases, similar to math operators. Their meaning depends on the traits of operands. They can be overloaded and overriden by defining a value with the operator identifier. The return trait of these functions must always be bool.

    with input=console, output=console:
    write "Enter X:"
    x = read Integer
    write "Enter Y:"
    y = read Integer

    if x > y:
        write "X is greater"
    elif x < y:
        write "Y is greater"
    else:
        write "They are equal"
ok
    equals\ def (left: Point, right: Point)->bool:
    return whether x left ?= x right and y left ?= y right
ok
