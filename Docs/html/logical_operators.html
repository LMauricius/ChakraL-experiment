    Usage

    Logical operators are used in condition checking for boolean alghebra. They can be overloaded and overriden by defining a value with the operator identifier. The return trait of these functions is always bool.

    with output=console:
    if health ?= 0.0 and extralives ?= 0:
        write "You died!"
